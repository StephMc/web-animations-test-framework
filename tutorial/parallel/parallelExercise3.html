Parallel Animation Gorup
Exercise 3 - Square (Challenge)

You have basically know all the things about parallel animation groups so let's get yourself prepared for some challenges.

  • In this exercise, you need to create an animation with the following requirements:
    1. Move 300px from left, colour changes from red to blue.
    2. Then move 400px from top, colour changes from blue to yellow.
    3. Then move 300px form right, colour changes from yellow to green.
    4. Then move 400px from bottom, colour changes from green to red.
    5. The animation should have ease-in effects as well as repeating itself 2 times.
    6. The total duration of animation is 5 seconds.
Show Solution
var group = new ParGroup([], {iterationCount: 2, timingFunction: new TimingFunction("ease-in")});
new Animation(document.querySelector("#a"), {left: "300px"}, 5.0, group);
new Animation(document.querySelector("#a"), {top: "400px"}, {duration: 5.0, startDelay: 5}, group);
new Animation(document.querySelector("#a"), {left: "0px"}, {duration: 5.0, startDelay: 10}, group);
new Animation(document.querySelector("#a"), {top: "0px"}, {duration: 5.0, startDelay: 15}, group);
new Animation(document.querySelector("#a"), {"backgroun-color": ["red", "blue", "yellow", "green", "red"]}, 20, group);
