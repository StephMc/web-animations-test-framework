Parallel Animation Group
Exercise 2 - Parent in Children

Beside including children into the the parallel group, there is another way to have children running parent's timing properties.

Firstly, you create a parallel group with no children in it (as child property is nullable) and only add the timing properties you need into the group. Then for each children you make, you add the parent into the child instead of timing properties:

var group = new ParGroup([], {duration: 2, iterationCount: 2});
new Animation(document.querySelector("#a"), {left: "100px"}, 2, group);

OR

new Animation(document.querySelector("#a"), {left: "100px"}, 2, new ParGroup([], {iterationCount: 2}));

For this exercise, make an animation that moves down 500px from top and changes colour from red to green then back to red in 5 seconds using parallel groups and iterates for 2 times.

Show Solution
new group = new ParGroup([], {duration: 5, iterationCount: 2});
new Animation(document.querySelector("#a"), {top: "500px"}, 5, group);
new Animation(document.querySelector("#a"), {"background-color": ["red", "green", "red"]}, 5.0, group);
