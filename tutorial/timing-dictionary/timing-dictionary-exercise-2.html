Exercise 2 - Start with Pause

Now you know how to create basic animations. We can now try to add some more timing properties into the animation

The same as exercise 1, let's make the animation move 300px from left and 400px from top diagonally. Then let the animation waits for 2 seconds before it will start. What property should we use this time?

Show Solution
new Animation(document.querySelector("#a"), {left: "300px", top: "400px"}, {startDelay: 2, duration: 5});
